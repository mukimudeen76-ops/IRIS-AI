# 🛡️ Security Best Practices

Security is the core pillar of the Autonomous Agent.

## The Trusted Operator Boundary
If the user commands IRIS to delete a file, and it does, that is a feature.
Vulnerabilities are defined as actions taken *without* the operator's consent.
